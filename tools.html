Painel de Avaliação dos Impactos de Mudanças - Termodinâmica

 

O PAIM-T é uma ferramenta desenvolvida para interpretar as condições climáticas futuras em cidades brasileiras. O objetivo principal é fornecer a pesquisadores, formuladores de políticas e profissionais uma ferramenta abrangente para avaliar e mitigar os impactos das mudanças climáticas em diferentes cidades brasileiras, oferecendo dados precisos para modelagem de conforto térmico e eficiência energética. A metodologia envolve a geração de arquivos Energy Plus Weather File (EPW) futuros, visualizando os resultados por meio de um dashboard de fácil utilização. Para mais detalhes, visite o Repositório Zenodo.
